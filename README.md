# House Price Prediction using Linear Regression

## 📌 Project Overview
This project was developed as part of the **SkillCraft Technology Machine Learning Internship (Task 1)**.

The objective is to build a Linear Regression model that predicts house prices based on:
- Square Footage (GrLivArea)
- Number of Bedrooms (BedroomAbvGr)
- Number of Bathrooms (FullBath)

---

## 📂 Dataset
House Prices - Advanced Regression Techniques Dataset

Target Variable:
- SalePrice

Input Features:
- GrLivArea
- BedroomAbvGr
- FullBath
